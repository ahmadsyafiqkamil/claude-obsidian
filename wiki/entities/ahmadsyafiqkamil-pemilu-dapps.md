# ahmadsyafiqkamil/pemilu-dapps

Sistem voting terdesentralisasi berbasis blockchain. Next.js 15 frontend + FastAPI backend + Solidity smart contracts (Foundry).

## Stack

| Layer | Teknologi |
|-------|-----------|
| Frontend | Next.js 15, React 19, TailwindCSS, RainbowKit, Wagmi |
| Backend | FastAPI, Python 3.x, Web3.py |
| Smart Contract | Solidity, Foundry |

## Fitur

- Voting period management (start/stop/emergency stop)
- MetaMask wallet integration via RainbowKit
- Admin dashboard + candidate management
- Real-time vote counting + result display
