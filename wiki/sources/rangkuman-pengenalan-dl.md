# Rangkuman Pengenalan Deep Learning

Modul penutup dari kursus **Belajar Fundamental Deep Learning** di Dicoding Indonesia. Merangkum seluruh materi dari taksonomi AI hingga arsitektur deep learning populer.

## Cakupan

- **Taksonomi AI**: AI → ML → NN → DL
- **Struktur ANN**: single perceptron, MLP (input/hidden/output layer)
- **Istilah kunci**: activation function (ReLU, sigmoid, tanh, softmax, leaky ReLU), loss function (MSE, cross-entropy), optimizer (SGD, RMSprop, Adam)
- **Forward propagation**: input → weighted sum → aktivasi → output
- **Backpropagation**: error → chain rule → gradient descent → update bobot
- **Arsitektur**: CNN, RNN, LSTM, GAN, Transformer, Autoencoder
