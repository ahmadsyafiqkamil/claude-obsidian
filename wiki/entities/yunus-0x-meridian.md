# yunus-0x/meridian

Repositori GitHub: https://github.com/yunus-0x/meridian (branch: experimental)

**490 stars** | JavaScript 100%

Platform AI untuk otomatisasi strategi liquidity provision di Meteora DLMM. Meridian memungkinkan user mendefinisikan agent LP yang berjalan secara otonom — memilih pool, mengelola posisi, dan mengeksekusi strategi tanpa intervensi manual.

## Konsep Utama

- **AI-powered LP agent** — autonomous trading agent untuk Meteora
- **High configurability** — user config yang sangat fleksibel untuk berbagai strategi
- **Dry run mode** — `npm run dev` untuk testing tanpa risiko real funds

## Peringatan dari Creator (Yunus)

> Kalian harus belajar dulu LP secara manual — strategi yang bagus itu gimana, filter coin yang bagus itu gimana — baru kalian bisa profit dari Meridian.

Meridian bukan shortcut untuk profit; ia adalah alat otomatisasi yang membutuhkan pemahaman fundamental LP terlebih dahulu.

## Disclaimer

Software disediakan as-is. Autonomous trading agent membawa risiko finansial nyata. Selalu mulai dengan dry run. Tidak pernah deploy modal lebih dari yang sanggup hilang.
