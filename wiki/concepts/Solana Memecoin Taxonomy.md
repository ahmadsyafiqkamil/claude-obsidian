# Solana Memecoin Taxonomy

Klasifikasi kategori memecoin Solana yang secara struktural tidak cocok untuk SOL-sided DLMM positions. Berdasarkan pengamatan empiris @EvilPanda.

## Kategori "No-Go"

| Kategori | Contoh | Masalah |
|----------|--------|---------|
| Political figure coins | Trump, Elon, Baron | Pump mingguan baru, selalu rug |
| Celebrity coins | Kanye | Hype-driven, no staying power |
| Justice coins | Iryna | Eksploitasi reputasi korban |
| Animal celebrities | TikTok animals | Viral → rug cycle pendek |
| DEV/BAGS coins | Generic dev token | OK jika dev aktif; red flag jika ghosting |

## Karakteristik Umum

- Dev memegang supply signifikan
- Tidak diketahui kapan dev memutuskan untuk cash in dan dump chart
- Chart dump bersifat slow rug (>-80% dalam 24h) tanpa bounce berarti
- Tidak ada fundamental atau use case di luar narasi

## Prinsip Screening

> Throw away that greed and just ignore these coins for good mental health.

Kategori-kategori ini adalah filter awal sebelum masuk ke screening kuantitatif [[GMGN Red-Flag Screening]]. Jika sebuah koin masuk ke salah satu kategori di atas, skip tanpa perlu cek metrics lebih lanjut.
