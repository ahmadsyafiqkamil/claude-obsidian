# Pengenalan Recurrent Neural Network

Modul dari kursus **Belajar Fundamental Deep Learning** di Dicoding Indonesia.

## Topik

- **Definisi RNN** — jaringan saraf untuk data berurutan (teks, audio, time series)
- **4 Tipe I/O RNN**: one-to-one, one-to-many, many-to-one, many-to-many
- **3 Jenis RNN**: Vanilla RNN, LSTM, GRU
- **4 Langkah kerja RNN**: input → aktivasi → pembaruan state internal → output
- **Aplikasi**: Siri, Google Translate, klasifikasi sentimen, terjemahan mesin
