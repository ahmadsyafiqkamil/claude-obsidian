# Pengantar Deep Learning

**Sumber**: Dicoding Indonesia — Belajar Fundamental Deep Learning, Tutorial 10069
**Diingest**: 2026-06-28

## Ringkasan

Artikel pengantar yang membahas definisi dan sejarah perkembangan deep learning dari perspektif akademi Dicoding.

## Konten Utama

### Sejarah

- 1943: McCulloch & Pitts — model jaringan saraf pertama
- 1950-60an: Frank Rosenblatt — perceptron
- 2006: Hinton, Bengio, LeCun — backpropagation efisien
- 2012: AlexNet memenangkan ImageNet — titik balik deep learning
- Era GPU + TensorFlow/PyTorch — akselerasi adopsi

### Definisi

Deep learning = kecerdasan buatan menggunakan [[Artificial Neural Networks]] (ANN) dengan banyak lapisan untuk belajar mandiri dari data besar.
