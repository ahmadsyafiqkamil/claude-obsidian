# jacksteamdev/obsidian-mcp-tools

MCP server providing vault access with semantic search and Templater execution.

## Key Features

- **Vault access**: via Obsidian's Local REST API plugin
- **Semantic search**: via Smart Connections plugin integration
- **Templater execution**: run Obsidian templates from AI with dynamic params
- **SLSA Level 3**: reproducible, signed builds (binary attestation)

## Dependencies

Requires three companion plugins: Local REST API + Smart Connections + Templater.

## Status

Seeking maintainers (open since Sep 2025). Active but maintenance uncertain.

## Standout

SLSA Level 3 attestation is unusually rigorous for an indie tool — supply chain integrity built in.

Templater execution from AI is a unique feature: AI can trigger vault automation, not just read/write notes.
