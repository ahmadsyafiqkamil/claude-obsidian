## Ringkasan

Artikel yang menjelaskan struktur arsitektur deep learning beserta jenis-jenis lapisan (layer) dan fungsi masing-masing.

## Konten Utama

### Struktur: Input → Hidden → Output

**Input Layer**: Menerima data, jumlah neuron = dimensi input (contoh: 32x32x3 = 3.072 untuk gambar RGB).

**Hidden Layers** (7 jenis):
1. Fully Connected / Dense
2. Convolutional (Conv)
3. Batch Normalization
4. Recurrent (RNN/LSTM/GRU)
5. Dropout
6. Pooling
7. Flatten

**Output Layer**: Jumlah neuron bergantung tugas — 1 neuron (biner), N neuron (multiclass).
